## 1. 本章定位

`00-总览` 是整套课程的总导航页。  
它不负责深入讲概念，而是先回答：

- 这套课程为什么这样设计
- 应该先学什么、后学什么
- 不同基础的学习者应该怎样进入

## 2. 适用人群

适合：

- 第一次进入这个仓库，不知道怎么开始的人
- 已经学过一些零散内容，想重新建立完整知识地图的人

## 3. 学习目标

学完这一章后，你应该能够：

1. 说清整套课程的主线结构
2. 理解为什么当前顺序是 `LLM -> Prompt -> 工具 -> RAG -> Agent -> 评测 -> 项目`
3. 根据自己的基础选择合适的阅读节奏

## 4. 推荐阅读顺序

1. [1.学习路线图.md](/Users/chenmingdong01/Documents/AI/agent/00-总览/1.学习路线图.md)
2. [2.知识地图.md](/Users/chenmingdong01/Documents/AI/agent/00-总览/2.知识地图.md)

建议先看路线图，再看知识地图。  
路线图解决“先后顺序”，知识地图解决“横向关系”。

## 5. 本章核心问题

进入正式学习前，建议先回答下面 4 个问题：

1. 我现在是在补基础，还是准备做项目
2. 我更缺概念理解，还是更缺工程落地
3. 我是要先做工具接入，还是先做知识问答
4. 我是否已经准备好进入 Agent，而不是只是被它吸引

## 6. 学完后的产出

建议至少完成：

1. 一份自己的学习顺序安排
2. 一张本仓库的知识关系图
3. 一份阶段性产出清单

## 7. 常见误区

- 一上来直接跳项目或 Agent
- 把 RAG、工具调用、Agent 当成彼此孤立的主题
- 只看热门概念，不建立完整主线

## 8. 总结与下一步

这一章的任务不是学知识点，而是先把路径走对。
