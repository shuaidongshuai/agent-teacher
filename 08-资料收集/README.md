## 1. 本章定位

`08-资料收集` 是辅助模块，不是主线章节。  
它解决的是：

**如何把“看到资料”变成“可转化的输入”，而不是只收藏链接。**

## 2. 适用人群

适合所有持续收集文章、课程、论文、仓库和案例的人。

## 3. 学习目标

学完这一部分后，你应该能够：

1. 用统一模板记录资料
2. 判断资料是否值得进入正式教学体系
3. 把外部输入转成仓库里的正式沉淀

## 4. 建议使用顺序

1. 用 [资料整理模板.md](/Users/chenmingdong01/Documents/AI/agent/08-资料收集/资料整理模板.md) 记录资料
2. 标注主题、阶段、适用人群和价值判断
3. 消化后再转写到正式章节
4. 关键概念同步补进 [09-术语表/README.md](/Users/chenmingdong01/Documents/AI/agent/09-术语表/README.md)

## 5. 学完后的产出

建议产出：

1. 一份高价值资料池
2. 一份待转写清单
3. 一批已消化并进入正式章节的资料

## 6. 常见误区

- 只收藏，不转写
- 不区分热点资料和长期有价值资料
- 资料和自己的课程体系脱节

## 7. 总结与下一步

资料收集的目标不是“越多越好”，而是为后续沉淀服务。

建议你把高价值资料继续转写到正式章节，并把概念同步补到 [09-术语表/README.md](/Users/chenmingdong01/Documents/AI/agent/09-术语表/README.md)。
