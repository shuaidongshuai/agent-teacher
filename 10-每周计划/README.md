## 1. 适用人群

这个目录适合边工作边学习，或者容易陷入“看了很多但没有输出”的学习者。

## 2. 学习目标

通过固定节奏，把学习从零散吸收变成可持续推进的过程。

## 3. 为什么要有每周计划

AI 学习很容易出现两个问题：

- 每周看了很多内容，但没有形成输出
- 每周都在追新东西，但核心主线没有推进

周计划的价值不是形式化打卡，而是帮助你持续回答：

- 本周主线是什么
- 本周真正产出了什么
- 本周卡点在哪里
- 下周应该怎么调整

## 4. 建议使用顺序

建议每周按下面顺序做一次：

1. 打开 [每周学习模板.md](/Users/chenmingdong01/Documents/AI/agent/10-每周计划/每周学习模板.md)
2. 写 1 个本周主题
3. 写 3 个明确目标
4. 周末回填输入、输出、卡点和下周计划

## 5. 高质量周复盘应该写什么

建议重点写 4 类内容：

1. 本周真正学会了什么
2. 本周真正做成了什么
3. 本周失败或卡住的原因是什么
4. 下周要减少什么分心项、加强什么主线

如果复盘只写“看了几篇文章”，通常价值不高。  
更有价值的是写：

- 自己理解变了什么
- 哪个项目推进了
- 哪个问题还没解决

## 6. 总结与下一步建议

每周计划的真正作用，是让你把长期学习拆成一段段可执行的小周期。

建议你把它和下面两个目录联动起来：

- `08-资料收集`
- `09-术语表`

这样每周就会形成一个闭环：

- 收资料
- 学主题
- 写术语
- 做项目
- 周复盘
