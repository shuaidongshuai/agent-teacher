# 项目练习清单

## 入门项目

1. 文本总结助手
2. 文章改写助手
3. 结构化信息抽取器

目标：

- 熟悉模型调用
- 练习 Prompt 设计
- 学会限制输出格式

## 进阶项目

1. 本地知识库问答系统
2. 带函数调用的日程/天气/搜索助手
3. PDF 阅读与问答工具

目标：

- 理解 RAG 全流程
- 理解工具调用如何接入真实世界
- 开始关注评测和容错

## Agent 项目

1. 研究助理 Agent
2. 能拆解任务的执行型 Agent
3. 多 Agent 协作 Demo
4. 学习任务拆解 Agent（见《学习任务拆解 Agent 实战》与 `agent-study-planner/`）

目标：

- 理解规划和执行不是一回事
- 理解 Agent 为什么容易失控
- 练习轨迹记录与复盘

## 复盘模板

每个项目都建议回答 4 个问题：

1. 它解决什么问题？
2. 它为什么需要 LLM，而不是普通规则？
3. 它最容易失败的地方在哪里？
4. 如果重做一次，我会优先改什么？
