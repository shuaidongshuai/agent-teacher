## 1. 适用人群

这个目录适合所有学习阶段的人，尤其适合觉得“概念好像懂了，但说不清楚”的学习者。

## 2. 学习目标

学会把常见术语转化成自己的语言，而不是只会背英文单词或行业黑话。

## 3. 为什么术语表很重要

很多人学习 AI 时有一种假象：

- 词都见过
- 文章也看过
- 但自己解释时说不清

这通常说明知识还停留在“识别”阶段，没有进入“内化”阶段。

术语表的价值就在于：

- 强迫你用自己的话重写概念
- 暴露理解漏洞
- 让不同主题之间建立连接

## 4. 建议阅读与维护顺序

建议按下面方式使用：

1. 先看 [核心术语.md](/Users/chenmingdong01/Documents/AI/agent/09-术语表/核心术语.md)
2. 每学完一个主题，就补 3 到 5 个新术语
3. 每隔一段时间回头重写旧定义，看看自己理解有没有升级

## 5. 怎样判断一个术语是否真的学会

可以用下面 3 个标准检查：

1. 你能不用原文，自己解释它是什么
2. 你能说清它解决什么问题
3. 你能举出它在项目里会怎样影响效果

如果三点都做不到，说明这个术语还没有真正掌握。

## 6. 总结与下一步建议

术语表看起来简单，但它对系统学习非常关键。  
因为它能把零散知识逐渐压缩成你自己的认知骨架。

建议你从现在开始，每学完一篇核心文档，就至少补 1 个术语定义。
