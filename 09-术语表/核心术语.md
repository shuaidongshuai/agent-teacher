# 核心术语

## Token

模型处理文本时使用的基本单位，不等于“一个汉字”或“一个单词”。

## Context Window

模型单次处理时能看到的上下文长度上限。

## KV Cache

大模型推理时保存历史 token 的 Key/Value 向量，避免自回归生成中反复计算整段历史上下文。

## Prefix Cache

对固定 system prompt、工具 schema、项目背景等稳定前缀进行缓存复用，用来降低重复请求的首 token 延迟和计算成本。

## HBM

High Bandwidth Memory，高带宽内存。现代 AI GPU 常用它承载模型权重、中间数据和 KV cache，容量与带宽会影响长上下文推理的吞吐、延迟和并发能力。

## Prompt

你给模型的输入说明，包括任务、约束、上下文和示例。

## Hallucination

模型生成了看起来合理但事实不正确的内容。

## RAG

先检索相关资料，再把资料连同问题一起交给模型生成答案。

## Function Calling

模型不直接输出最终答案，而是先输出结构化调用请求，让程序去执行工具。

## Agent

一个能围绕目标持续感知、决策、调用工具并执行多步任务的系统。
