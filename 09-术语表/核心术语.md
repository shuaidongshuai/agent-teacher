# 核心术语

## Token

模型处理文本时使用的基本单位，不等于“一个汉字”或“一个单词”。

## Context Window

模型单次处理时能看到的上下文长度上限。

## Prompt

你给模型的输入说明，包括任务、约束、上下文和示例。

## Hallucination

模型生成了看起来合理但事实不正确的内容。

## RAG

先检索相关资料，再把资料连同问题一起交给模型生成答案。

## Function Calling

模型不直接输出最终答案，而是先输出结构化调用请求，让程序去执行工具。

## Agent

一个能围绕目标持续感知、决策、调用工具并执行多步任务的系统。
