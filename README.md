# LLM 与 Agent 学习资料库

这是一个面向系统学习的资料整理目录，目标不是单纯堆链接，而是把知识、笔记、项目、复盘放进同一个结构里。

## 建议学习顺序

1. `01-LLM基础`：先理解大模型基本概念和工作方式。
2. `02-Prompt工程`：学习如何把任务说清楚。
3. `03-RAG`：学习检索增强生成的核心流程。
4. `04-工具调用与函数调用`：理解模型如何连接外部工具。
5. `05-Agent`：理解 Agent 的结构、规划、记忆与执行。
6. `06-评测与优化`：学会评估效果，而不是只看“能不能跑”。
7. `07-项目实战`：把前面的知识串起来做项目。
8. `11-模拟面试`：用问答和场景题检查自己是否真的学会。

## 目录说明

- `00-总览`：学习路线图、知识地图、总目标。
- `01-LLM基础`：Transformer、Token、上下文窗口、推理、微调等。
- `02-Prompt工程`：提示词设计、任务拆解、结构化输出。
- `03-RAG`：切分、Embedding、召回、重排、上下文构建。
- `04-工具调用与函数调用`：函数调用、API 编排、工具协议。
- `05-Agent`：规划、工具使用、记忆、工作流、多 Agent。
- `06-评测与优化`：离线评测、在线反馈、幻觉、延迟、成本。
- `07-项目实战`：从简单问答助手到 Agent 应用。
- `08-资料收集`：课程、文章、论文、仓库与视频整理区。
- `09-术语表`：把关键概念翻译成你自己的语言。
- `10-每周计划`：学习节奏、打卡、阶段复盘。
- `11-模拟面试`：综合问答、场景题、项目复盘题与参考答案。

## 使用方法

你以后可以按下面方式维护：

- 新看到一篇资料，先放到 `08-资料收集`
- 消化之后，把自己的理解写回对应主题目录
- 每学完一个主题，在 `09-术语表` 补关键概念
- 每周在 `10-每周计划` 做一次回顾

## 一个实用原则

学习 LLM 和 Agent，不要只收藏资料。建议保持 3 条并行线：

- 概念线：知道它是什么、为什么存在
- 工程线：知道它怎么搭建、怎么调试
- 实战线：知道它在真实应用里为什么会失败
